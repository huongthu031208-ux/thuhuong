Ngôn ngữ truy vấn SQL

Giới thiệu Cú pháp Ví dụ

📘 Giới thiệu

SQL (Structured Query Language) là ngôn ngữ dùng để quản lý và truy vấn cơ sở dữ liệu quan hệ. Đây là công cụ quan trọng trong việc lưu trữ và xử lý dữ liệu.

Xem tiếp →

⚙️ Câu lệnh cơ bản

  • SELECT: Lấy dữ liệu
  • INSERT: Thêm dữ liệu
  • UPDATE: Cập nhật dữ liệu
  • DELETE: Xóa dữ liệu

💡 Ví dụ SQL

SELECT * FROM HocSinh;

INSERT INTO HocSinh (MaHS, TenHS)
VALUES ('HS01', 'Nguyen Van A');

UPDATE HocSinh
SET TenHS = 'Tran Van B'
WHERE MaHS = 'HS01';

DELETE FROM HocSinh
WHERE MaHS = 'HS01';
